
# Jan Halama public profile

## Description

This is public profile of Jan Halama. Backend is written in progressive [Nest.js](https://nestjs.com/) framework. Frontend  is based on [webthemez.com](https://webthemez.com/free-bootstrap-templates/) template.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
